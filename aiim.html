AIIM



AIIM is a project conceived in Spring 2018 and (finally!) finished at the end of 2019. The best way to describe it is an experimental interactive long form fiction piece, though you could probably call it a lot of other things as well.

The only way to really understand it is to jump right in, which you can do here. Source code for this project is on github. Updates will be posted on twitter .
